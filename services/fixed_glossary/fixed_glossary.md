正在做 AKB48 相关节目、MC、综艺和直播切片字幕的日文→中国大陆简体中文翻译。目标不是正式学术翻译，而是让中文字幕自然、好读、辨识度高，同时不要乱创不适合简体中文语境的译名。核心目标是：让中文观众尽可能接收到日本观众看到成员名、团名、节目名和梗时能感受到的资讯量。

## 档案结构：talents / others

`fixed_glossary.json` 分两大类：

```jsonc
MappingBlock = { "jp": [str, ...], "zh": str }

{
  "talents": [
    {
      "group": MappingBlock,            // 可选：组合名；solo 艺人省略
      "members": [MappingBlock, ...]    // 至少一名成员
    }
  ],
  "others": [MappingBlock, ...]         // 节目／单元／品牌／术语／作品等
}
````

* `talents`：出演者／团体／组合。团体或组合用一个 unit 包起来，`group` 放团体名或组合名，`members` 放成员。
* `others`：节目名、单元名、品牌、平台、搞笑术语、作品名等非人物专名。
* 把组合和成员包在一起，是为了解决裸成员名模糊问题，例如 `きむ`、`リリー`、`ガク`、`ノブ`、`せいや`。命中任一成员或组合名时，整个 unit 可以一起喂给 pre-pass，让模型知道上下文。

## jp[] 使用原则

`jp[]` 只放组合／人物最原始、正确的 source 来源拼写。

以下变体由 `services\gemini\fixed_glossary.py <_normalize_jp>` 处理，不要重复收进 `jp[]`：

* NFKC：全形→半形、相容字
* 片假名 → 平假名
* 去除长音符、波浪、中点、空白、Tab、换行、常见标点
* casefold：大小写折叠

好例：

```jsonc
{ "jp": ["熊元プロレス"], "zh": "熊元摔角" }
{ "jp": ["ほんこん"], "zh": "Honkon" }
```

坏例：

```jsonc
{ "jp": ["熊元プロレス", "熊元 プロレス", "熊元　プロレス"], "zh": "熊元摔角" }
{ "jp": ["ほんこん", "ホンコン"], "zh": "Honkon" }
{ "jp": ["ホンコン", "香港"], "zh": "Honkon" } // 香港可能真的是地名
```

ASR 相似音或严重走钟，不靠 `jp[]` 硬塞猜测，交给 full 模式与 LLM 根据音讯／画面确认。

## 专名翻译总原则

专名包含：节目名、单元名、搞笑团体名、艺人名、艺名、作品名、品牌名、梗名。

整体优先顺序：命中第一个适用者即停。

### 1. 有稳定大陆／中文译名就用既有译名

只在能确认常见、稳定、官方或中文圈已广泛使用时采用。不要自己发明看起来很像中文节目名、但没有使用基础的创意译名。

例：

| 日文             | 简中    |
| -------------- | ----- |
| ロンドンハーツ / ロンハー | 男女纠察队 |
| 逃走中            | 全员逃走中 |
| るろ剣            | 神剑闯江湖 |
| 南海キャンディーズ      | 南海甜心  |
| ネプチューン         | 海王星   |

### 2. 人名／艺名可回推汉字时，要看该汉字是不是名称核心

若假名只是本名、姓氏、名字的假名表记，或中文圈已有稳定汉字用法，可用简体中文汉字。

例：

| 日文                | 简中     |
| ----------------- | ------ |
| 河井ゆずる / 河井譲       | 河井让    |
| お见送り芸人しんいち / 上野晋一 | 送别艺人晋一 |
| イワクラ / 岩仓美里       | 岩仓     |
| みなみかわ / 南川聡史      | 南川     |
| 月亭方正              | 月亭方正   |

若假名部分可明确回推姓名汉字，而且该汉字是艺名来源、梗点或辨识重点，保留汉字；其余外来语／假名部分再用英文或罗马字。

例：

| 日文      | 简中     |
| ------- | ------ |
| きむらバンド  | 木村Band |
| かが屋     | 加贺屋    |
| クワバタオハラ | 桑波田小原  |

但若艺名已经刻意片假名化、暱称化、角色化，即使知道本名汉字，也不一定回推成本名汉字；通常保留艺名读音，用罗马字。

例：

| 日文   | 简中     |
| ---- | ------ |
| ユースケ | Yusuke |
| ガク   | Gaku   |
| ケツ   | Ketsu  |
| きょん  | Kyon   |
| おたけ  | Otake  |
| きむ   | Kimu   |

可以翻节目名、单元名、团名、艺名中的描述性成分。

但如果翻完会变成普通名词、食品名、物品名、普通句子、店名，导致不像艺人／团体名，就不要硬翻，改用罗马字或既有英文表记。

### 4. 来源可解析的名称，优先转译「名称结构」与资讯量

如果团名、艺名或节目名的来源可解析，优先保留原名的命名结构，让中文观众也能看出原本的梗、语意、汉字核心、成员姓名、地名或外来语成分。

这不是乱创译名，而是把日本观众看得懂的命名资讯，转成中文观众也能理解的形式。

处理方式不一定是全中文，也不一定是全罗马字。可以依原名结构选择：

- **中文化**：来源语意明确，翻成中文后仍像专名。
- **半翻半保留外来语**：原名本身是「中文可理解成分 + 外来语／英文」的混成。
- **保留汉字核心 + 罗马字／英文**：原名有姓名、地名、汉字来源，但部分成分不适合翻中文。
- **回推姓名／地名／典故汉字**：来源明确且是名称核心时，不要只罗马字化。

判断重点：

- 中文观众是否能从译名看出原名的命名意图。
- 翻完是否仍像专名，而不是普通句子、商品名、物品名或解释文。
- 是否保留原名的结构比例，而不是一律全中文或全英文。
- 若只罗马字化会让中文观众完全看不出来源，应优先考虑中文化或半中文化。
- 外来语成分不是一律保留英文，也不是一律翻中文。若中文里已有稳定、自然、且不失专名感的对应词，可以中文化；若翻成中文会太解释、太抽象、太普通，或失去原本外来语质感，则保留英文／罗马字。

例：

| 日文 | 简中 | 理由 |
|---|---|---|
| かもめんたる | 海鸥Mental | `mental` 翻成心理／精神太解释，保留外来语质感 |
| 真空ジェシカ | 真空洁西卡 | `Jessica` 是人名，中文有稳定译名洁西卡 |
| 鬼越トマホーク | 鬼越战斧 | `Tomahawk` 有稳定自然的中文对应 |
| チョコレートプラネット | 巧克力星球 | 外来语全中文化后仍像团名 |
| バナナマン | 香蕉人 | 中文化后自然且像团名 |
| 空前メテオ | 空前Meteor | 保留汉字核心 + 外来语 |
| 金属バット | 金属Bat | `Bat` 若译成球棒，太像普通物品 |
| カベポスター | 墙壁Poster | `カベ` 中文化，`Poster` 保留外来语 |
| 9番街レトロ | 第九街Retro | `9番街` 中文化，`Retro` 保留外来语 |
| 相席スタート | 相席Start | `Start` 译成开始会太像普通片语 |
| ロングコートダディ | 长外套Daddy | 来源画面是长外套大叔，保留 `Daddy` 艺名感 |
| ダブルヒガシ | Double东 | 保留 `Double + 东` 的原名结构 |
| ダイタク | 大拓 | 来自成员名字 `大 + 拓`，比 `Daitaku` 更有资讯量 |
| かが屋 | 加贺屋 | 来自成员姓氏 `加贺 + 贺屋` |
| クワバタオハラ | 桑波田小原 | 来自成员姓氏 |
| ランジャタイ | 兰奢待 | 可回推典故汉字 |
| ガクテンソク | 学天则 | 可回推典故来源 |
| ひつじねいり | 羊礻入 | 保留拆字来源 |
| 家族チャーハン | 家族炒饭 | 语意翻完仍像怪团名 |
| 例えば炎 | 例如炎 | 保留原名怪感，短且像团名 |
| 素敌じゃないか | 这不是很棒吗 | 句子型团名，翻语气比罗马字好 |

不建议只保留读音的例子：

| 日文 | 不建议 | 较佳 |
|---|---|---|
| かもめんたる | Kamomental | 海鸥Mental |
| カベポスター | Kabe Poster | 墙壁Poster |

| ひつじねいり | Hitsujineiri | 羊礻入 |

但不要过度中文化。若翻完像普通物品、商品名、硬翻外号或中文自创名，应改用英文／罗马字或半保留形式。

| 日文 | 不建议 | 较佳 |
|---|---|---|
| モグライダー | 鼹鼠骑士 | Mogu Rider |
| サルゴリラ | 猴猩猩 | Sarugorilla |

| 松井ケムリ | 松井烟 | 松井Kemuri |
| 髙比良くるま | 髙比良车 | 髙比良Kuruma |

### 5. 平假名名称比片假名更积极检查中文化可能

若平假名或片假名名称可依第 4 点回收来源资讯，优先照第 4 点处理；只有来源不明、中文化后不像专名时，才转向罗马字／英文表记。

例：

| 日文      | 简中     |
| ------- | ------ |
| みなみかわ   | 南川     |
| かが屋     | 加贺屋    |
| ひつじねいり  | 羊礻入    |
| もう中学生   | 已是中学生  |
| 素敌じゃないか | 这不是很棒吗 |
| かまいガチ   | 镰鼬来真的  |

但这不是硬规则。平假名如果是暱称化艺名，仍可用罗马字：

| 日文   | 简中     |
| ---- | ------ |
| きょん  | Kyon   |
| さすけ  | Sasuke |
| たくろう | Takuro |

### 6. 片假名名称优先判断外来语、英文、品牌化表记

片假名常是外来语、英文、品牌化表记或音感型团名。若是明确英文／外来语，优先用英文或常见表记。

例：

| 日文            | 简中                            |
| ------------- | ----------------------------- |
| ロングコートダディ     | Long Coat Daddy               |
| ザ・マミィ         | The Mommy                     |
| ジャングルポケット     | Jungle Pocket                 |
| ニューヨーク        | New York                      |
| コロコロチキチキペッパーズ | Koro Koro Chiki Chiki Peppers |
| コロチキ          | KoroChiki                     |
| トット           | Totto                         |
| ミカボ           | Mikabo                        |

但片假名若有明确汉字／典故来源，可以中文化：

| 日文     | 简中  |
| ------ | --- |
| ランジャタイ | 兰奢待 |
| ガクテンソク | 学天则 |
| イワクラ   | 岩仓  |

### 7. 罗马字优先用官方／常见英文表记，不要机械逐音转写

已有官方、常见、资料库惯用英文表记时，优先采用。固定大小写与空格，组合名与节目名也要一致。

例：

| 日文          | 简中                 |
| ----------- | ------------------ |
| ダウ90000     | Daw90000           |
| パンプキンポテトフライ | Pumpkin Potato Fry |
| ダウンタウン      | DOWNTOWN           |
| ダウンタウンDX    | DOWNTOWN DX        |
| 水曜日のダウンタウン  | 星期三的DOWNTOWN       |
| さまぁ〜ず       | SUMMERS            |
| ジャルジャル      | JARUJARU           |
| ラランド        | LALANDE            |
| バッテリィズ      | Batterys           |

### 8. 不保留假名，唯一例外是日文文字视觉双关

翻中文或罗马字都优先于保留假名；对看中文字幕、不懂日文的观众，罗马字／中文的辨识度一律高于假名。唯一例外是名称高度依赖日文文字视觉（字形）双关、罗马字化会直接毁掉该梗时才保留原文。单纯语音双关不算（罗马字已保留读音），辨识度较低也不算；其余一律走 tier 4 罗马字，或前面 tier 的中文／汉字回收。

不要预设保留成假名：

```text
カカロニ
ケムリ
すがちゃん
や団
なえなの
ほんこん
かまいガチ
```

## 专名翻译和一般字幕翻译要分开

一般字幕句子中，不要擅自补「我／你／他／她／我们／大家」等日文省略主词。

但节目名、艺名、梗名是固定名词，可以为了保留语气做固定化翻译。

例：

| 类型   | 日文          | 简中             |
| ---- | ----------- | -------------- |
| 一般句子 | 知らんけど       | 不一定要补成「我是不知道啦」 |
| 节目名  | かまいたちの知らんけど | 镰鼬的我是不知道啦      |

## 不要做的事

* 不要把片假名艺名直接照意思硬翻。

  * 坏：`松井ケムリ` → `松井烟`
  * 好：`松井ケムリ` → `松井Kemuri`

* 不要擅自帮假名人名补汉字。

  * 坏：`きょん` → `恭`
  * 好：`きょん` → `Kyon`

* 不要把节目名翻成很像中文节目、但没有使用基础的创意标题。

  * 坏：`かまいガチ` → `镰鼬真挑战`
  * 好：`かまいガチ` → `镰鼬来真的`

* 不要把搞笑团体名硬翻成普通名词。

  * 坏：`モグライダー` → `鼹鼠骑士`
  * 好：`モグライダー` → `Mogu Rider`

* 不要把字面有意思的艺名翻成普通句子。

  * 坏：`おいでやす小田` → `欢迎光临小田`
  * 好：`おいでやす小田` → `Oideyasu小田`

* 不要把暱称化艺名翻成自创中文暱称。

  * 坏：`おたけ` → `小武` / `阿武`
  * 好：`おたけ` → `Otake`

* 不要在中文字幕里大量留下日文假名。

  * 翻成中文后仍像专名，才中文化。
  * 否则优先罗马字／常见英文表记。
  * 假名仅限日文文字视觉双关才保留，语音双关与辨识度都不算，其余一律罗马字／常见英文表记。

* 不要把「组合＋成员」拼成一条 `jp` 别名。

  * 组合关系用 `talents` 结构表达。
